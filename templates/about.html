About this Project

NCT car checker tool was developed as part of my Masters Thesis Project and to know about NCT history of car brands. This tool was developed by using the NCT yearly dataset from 2015-19. This tool would help public to know which brands of cars performed better, there by maximizing chances of passing NCT. Also to query NCT vehicle testing data and to give facts and figures. This tool was designed using python Django framework and object-oriented programming. Python libraries such as pandas, were used to read the dataset.
